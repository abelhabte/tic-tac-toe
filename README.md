# tic-tac-toe

A tic-tac-toe game in the terminal.

Each position entry is described as
below.

| Header 1 | Header 2 | Header 3 |
|----------|----------|----------|
| 1,1      | 1,2      | 1,3      |
| 2,1      | 2,2      | 3,3      |
| 3,1      | 3,2      | 3,3      |

| 1,1      | 1,2      | 1,3      |
|----------|----------|----------|
| 2,1      | 2,2      | 3,3      |
| 3,1      | 3,2      | 3,3      |
